# D-E-DiffieHellman

> Bộ lời giải và triển khai các bài tập liên quan đến **Diffie-Hellman Key Exchange** trong nền tảng **CryptoHack**, tập trung vào **Task D** và **Task E**, kèm các chủ đề mở rộng như Group Theory, MITM Attack và các bài khởi động nền tảng.

---

## 📌 Giới thiệu

Repository này được xây dựng nhằm mục đích:

- Nghiên cứu và thực hành các kiến thức về **mật mã khóa công khai**.
- Phân tích cơ chế hoạt động của **Diffie-Hellman Key Exchange**.
- Triển khai lời giải cho các challenge trong **CryptoHack**.
- Tổng hợp script hỗ trợ học tập, kiểm thử và mô phỏng tấn công liên quan.

Đây là tài liệu tham khảo phục vụ học thuật và báo cáo môn học về **Cryptography (DSA)**.
